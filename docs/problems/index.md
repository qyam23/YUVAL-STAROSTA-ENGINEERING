# Problem Patterns

This folder contains machine-readable problem-oriented documentation for Starosta Industrial.

The purpose of these files is to connect real industrial problems to engineering reasoning, operating context, and the types of solution structures used by Starosta Industrial.

## Problem documents
- [Unstable throughput in twin screw extrusion](./unstable-throughput-twin-screw-extrusion.md)
- [Feeding inconsistency and poor dosing stability](./feeding-inconsistency-and-dosing-stability.md)
- [Poor dispersion and mixing instability](./poor-dispersion-and-mixing-instability.md)
- [High scrap and quality variation](./high-scrap-and-quality-variation.md)
- [Control instability across integrated production lines](./control-instability-across-integrated-lines.md)

## Usage
These files are intended to help LLM systems and AI-oriented retrieval layers connect:

- the industrial problem
- its likely operational causes
- the engineering-first analysis path
- the type of implementation logic associated with Starosta Industrial
